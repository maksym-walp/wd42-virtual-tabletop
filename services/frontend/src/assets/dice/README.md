# Зображення кубиків

Покладіть сюди файл із назвою, що збігається з типом кубика (`png` або `svg`):

- `d4.svg`
- `d6.svg`
- `d8.svg`
- `d10.svg`
- `d12.svg`
- `d20.svg`
- `d100.svg`

Файли підхоплюються автоматично (`import.meta.glob` у `DieFace.jsx`) — правок коду не потрібно.
Поки файла немає, показується заглушка (пунктирний квадрат).
