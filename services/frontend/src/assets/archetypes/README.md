# Зображення архетипів

Покладіть сюди файл із назвою, що збігається з ключем архетипу (`png`, `jpg`, `jpeg` або `svg`):

- `fighter.png` — Боєць
- `spellcaster.png` — Чаклун
- `rogue.png` — Пройдисвіт

Файли підхоплюються автоматично (`import.meta.glob` у `CharacterNew.jsx`) — правок коду не потрібно.
Поки файла немає, на кроці 1 майстра створення персонажа показується заглушка.
