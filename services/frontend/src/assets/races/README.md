# Зображення народів

Покладіть сюди файл із назвою, що збігається з ключем народу (`png`, `jpg`, `jpeg` або `svg`):

- `human.png` — Людина
- `gnome.png` — Гном
- `dwarf.png` — Дворф
- `elf.png` — Ельф
- `sangvi.png` — Санґви
- `nephilim.png` — Нефілім
- `other.png` — Інший народ

Файли підхоплюються автоматично (`import.meta.glob` у `CharacterNew.jsx`) — правок коду не потрібно.
Поки файла немає, на кроці 1 майстра створення персонажа показується заглушка.
